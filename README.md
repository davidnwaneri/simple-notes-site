# Simple Notes Site

Landing page for the Simple Notes App, featuring app information and account deletion instructions.

## Live Site

Visit the live site at: `https://davidnwaneri.github.io/simple-notes-site/`

## Enabling GitHub Pages

To deploy this site:

1. Go to repository Settings
2. Navigate to Pages section
3. Under "Source", select the branch to deploy (e.g., `main` or `copilot/add-landing-page-for-notes-app`)
4. Select "/ (root)" as the folder
5. Click Save

The site will be live within a few minutes.

## Local Development

To view locally:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.
